## Getting Started
1. Clone the repository:
   ```sh
   git clone https://github.com/yasuo72/finnsathi-fresh-.git
   ```
2. Add a `.env` file in the root directory with your backend API URL:
   ```env
   BACKEND_BASE_URL=https://your-backend-url.com
   ```
3. Run the app:
   ```sh
   flutter pub get
   flutter run
   ```
